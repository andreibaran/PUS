- user guide -


Register

When accessing the application, the user will first be led to the presentation page, with the login and register functionalities.

The in the register section, the user will create a new account by providing his name, an username and a password in the correspoding fileds

After creating the account, the user can login on his page by inserting his credentials in the fields from the top right of the page.


Add a device

Once he manages to access his account, the user can couple with one ore more mobile devices.


Add a device

After the device is added, by clicking on its corresponding icon, the user will be redirect to its page, from where he can update the information provided by the phone's sensors and also perform actions on them.


Change the default account settings

The user has the option to manually change the font size of the application from the Settings section from the menu
